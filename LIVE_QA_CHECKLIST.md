# Live URL 최종 확인 체크리스트

GitHub Pages 주소가 열린 뒤 아래만 확인하면 됩니다.

## A. 첫 화면

- [ ] 사이트 제목이 보인다.
- [ ] Book1~Book5가 보인다.
- [ ] Progress 표시가 보인다.
- [ ] 모바일에서 메뉴 버튼이 보인다.

## B. Book별 데이터 로딩

- [ ] Book1 클릭 → R1~R6 확인
- [ ] Book2 클릭 → R7~R16 확인
- [ ] Book3 클릭 → R17~R22 확인
- [ ] Book4 클릭 → R23~R33 확인
- [ ] Book5 클릭 → R34~R42 확인

## C. 검색 테스트

검색창에 하나씩 입력:

- [ ] `duration`
- [ ] `FCFE`
- [ ] `OAS`
- [ ] `ETF`
- [ ] `Ethics`
- [ ] `residual income`
- [ ] `Monte Carlo`

기대 결과:
- 관련 LOS가 나온다.
- 검색 결과 클릭 시 해당 LOS로 이동한다.

## D. 퀴즈 테스트

- [ ] 아무 LOS에서 `정답 보기` 클릭
- [ ] 정답과 해설이 보인다.
- [ ] 버튼이 `정답 숨기기`로 바뀐다.
- [ ] 다시 누르면 접힌다.

## E. 저장 기능 테스트

- [ ] 오답노트/메모에 문장 입력
- [ ] 다른 Reading 이동
- [ ] 다시 돌아와도 메모가 남아 있다.
- [ ] 완료 체크를 누르면 진행률이 바뀐다.
- [ ] 새로고침해도 완료 체크가 유지된다.

## F. 모바일 테스트

- [ ] 휴대폰에서 열었을 때 카드가 화면 밖으로 삐져나가지 않는다.
- [ ] 표가 읽을 수 있다.
- [ ] 메뉴가 열린다.
- [ ] 퀴즈 버튼이 눌린다.
